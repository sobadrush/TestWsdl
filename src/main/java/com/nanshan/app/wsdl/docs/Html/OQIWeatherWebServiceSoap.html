OQIWeatherWebServiceSoap

SOAP Version: 1.1

Main class which you can use to connect to your web service

Important settings

  • Timeout - the timeout for network operations, in milliseconds. By default: 60 sec.
    Example:
                        
OQIWeatherWebServiceSoap service = new OQIWeatherWebServiceSoap(null,"http://www.webxml.com.cn/WebServices/WeatherWebService.asmx",30000);
service.getSupportCity();
  • Url - the web service address. By default it is retrieved from the wsdl file
    Example:
                        
OQIWeatherWebServiceSoap service = new OQIWeatherWebServiceSoap(null,"http://www.webxml.com.cn/WebServices/WeatherWebService.asmx");
service.getSupportCity();
  • httpHeaders - allows you to add custom headers to the http requests.
    Example:
                        
OQIWeatherWebServiceSoap service = new OQIWeatherWebServiceSoap();
service.httpHeaders.add( new HeaderProperty("Authorization","Basic QWxhZGRpbjpPcGVuU2VzYW1l"));
service.getSupportCity();
  • Enable logging - if you enable logging, all requests/responses will be write to the console (for debugging purposes)
    Example:
                        
OQIWeatherWebServiceSoap service = new OQIWeatherWebServiceSoap();
service.enableLogging=true;
service.getSupportCity();
  • createClassesForAny - if true, detect a strong types in any collections
    Example:
    service.createClassesForAny=true;
